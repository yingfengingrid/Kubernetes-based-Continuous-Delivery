# Kubernetes-based-Continuous-Delivery

## Install

* Create a virtualenv and source

* run `make install`

   `step1.sh`

### Running app

#### Three modes:

1. Standalone:  `python app.py`
2. Run in Docker:  `./run_docker.sh`
3. Run in Kubernetes:  `./run_kubernetes.sh`

#### Prediction

Run  `make_prediction.sh`
